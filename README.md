# 健康管理アプリ（Health Log）

## 概要
このアプリは、日々の **睡眠時間・運動時間・体調・食事内容** を記録し、  
その推移をグラフで可視化できるシンプルな健康管理アプリです。

Streamlit を用いてUIを構築し、データの保存には Supabase を利用しています。  
そのため、アプリを停止・再起動しても記録したデータは消えず、継続的に管理できます。

---

## アプリの機能

### 1. 健康データの記録
以下の項目を入力して保存できます。

- 日付  
- 睡眠時間（時間）
- 運動時間（分）
- 体調（1〜5のスライダー）
- 食事内容（自炊／外食／脂質多め／野菜中心）

### 2. データの永続保存
- Supabase のデータベースを使用
- アプリを再起動してもデータは保持される

### 3. サマリー表示
これまでの記録から以下の平均値を自動計算・表示します。

- 平均睡眠時間
- 平均運動時間
- 平均体調スコア

### 4. グラフによる可視化
- 睡眠時間の推移（推奨睡眠時間 7時間の目安線付き）
- 運動時間の推移
- 体調スコアの推移

### 5. 記録一覧と削除機能
- 過去の記録を一覧表示
- 不要な記録はボタン1つで削除可能

## 注意事項

- Supabase の Row Level Security（RLS）はオフにしています
- データは学習・課題用途を想定しています
- 個人情報や機密情報の入力は行わないでください
